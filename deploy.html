📖 Usage Guide

  1. Load Sample Data: Click "Load Sample" to start with pre-loaded crash data
  2. Train Model: Click "Train Model" to train the ML algorithm on your data
  3. Make Prediction: Click "Predict Next" to get a single prediction
  4. View Results: See the predicted multiplier and confidence score
  5. Batch Predictions: Click "Batch (5x)" to predict the next 5 crashes

The Crash Predictor uses a Random Forest Machine Learning model to analyze patterns in historical crash data.

  • 📊 It learns from previous crash multipliers
  • 🔍 Identifies patterns and trends in the data
  • 🎯 Makes predictions based on the last 5 crashes
  • 📈 Improves accuracy with more historical data
  • 💯 Provides confidence scores for each prediction
  • ✅ Use at least 20-30 crash data points for better accuracy
  • ✅ More data = better predictions
  • ✅ Monitor confidence scores - higher is better
  • ✅ Retrain the model when you have new data
  • ✅ Batch predictions are chained - each uses the previous prediction
  • ❌ Don't rely on predictions alone - use them as guidance
